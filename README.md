# WSD-python
Web Services for Devices (WSD) tools and utilities for cross platform support.


## wsd-discovery.py
```
usage: wsd-discovery.py [-h] [-D] [-T T]

WSD Discovery Scanner

optional arguments:
  -h, --help  show this help message and exit
  -D          Enable debug
  -T T        Timeout
```
